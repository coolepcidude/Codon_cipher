About This Cipher

The Codon Cipher is a custom-made encryption tool inspired by the genetic codon table. It maps characters (letters, symbols, and numbers) to unique 3-digit codes based on a 4x4x4 grid system.

This website lets you encode and decode messages using this codon system. It's meant for fun, creativity, and learning — not for real-world security.

How it works

  • Each character is matched with a codon like 111, 234, etc.
  • Encoding converts your message to codons.
  • Decoding brings it back to readable text.
  • Input is case-insensitive (A = a).
